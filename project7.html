Project Visual

Visual design

Met dit project heb ik gewerkt aan veel verschillende opdrachten die te maken hebben met visual design in de breedste zin van het woord. Ik ben in aanraking gekomen met allerlei aspecten van het vakgebied; ontwikkelen van mijn blik, competenties, skills en argumentatie. Doordat dit project bestond uit erg veel losse opdrachten heb ik besloten om deze niet op deze website te zetten. Wel kunnen de resultaten van een workshop Illustrator hieronder bekeken worden.

April 2021 (CMD jaar 2)
