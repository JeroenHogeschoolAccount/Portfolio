Minor VID

Korte introductie

Tijdens deze minor heb ik me bezig gehouden met visual interface design. De vakken: Grid & Kleur, Typografie, Beeldtaal, Interface & Beweging, Ontwerpen 1, Interface & Interactie, Webtypografie en Ontwerpen 2.

Februari 2022 (CMD jaar 3)

Grid & Kleur

Door Grid & Kleur ben ik me vooral bewust geworden dat de juiste kleuren kiezen erg belangrijk is om de juiste sfeer en emotie over te brengen. Ik zal bij toekomstige projecten daarom ook mijn kleuronderzoek-document erbij halen. Verder heb ik veel geleerd over grid, visuele hiërarchie en flow in compositie (zie documentatie hier beneden). Deze kennis heb ik goed kunnen verwerken in een interface voor een groot scherm. Dit was één van de dingen dat ik graag wilde leren bij deze minor, aangezien ik normaal gesproken vaak een interface ontwerp voor small screen (smartphone).

Typografie

Door het vak Typografie en het artikel "Type That Really Talks" ben ik op een andere manier gaan kijken naar letters. Bij het zien van letters op posters, gebouwen etc. denk ik nu automatisch na waarom dat specifieke font is gekozen. Ook bij het uitkiezen van een font voor bijvoorbeeld een interface kijk ik niet enkel naar 'wat is mooi', maar naar het 'gehele plaatje'. Doordat er ongelooflijk veel fonts op het internet beschikbaar zijn kan een keuze maken best lastig zijn. Ik wist eerst niet goed waar ik op moest letten. Dankzij dit vak zal het kiezen van een font gemakkelijker gaan.

Beeldtaal

Kennis opgedaan van Semiotiek, Retorica en Gestalt. Met deze kennis kan ik betekenisvolle visuals maken en daarmee communiceren met de bijbehorende doelgroep.

Interface & Beweging

Bij dit vak kennis opgedaan over animatie en het ontwikkelen van een originele stijl. Animeren was compleet nieuw voor mij, dus het onder de knie krijgen van Adobe Animate was even pittig. Uiteindelijk meerdere animaties met geluid weten te produceren en deze verwerkt in een interface. Ik vond het een erg leuk vak waarvan ik de opgedane kennis zeker in de toekomst kan gaan gebruiken. In mijn vrije tijd wil ik me verder gaan ontwikkelen in het gebruik van Adobe Animate. Ook de kennis over motion graphics was erg nuttig, waarmee ik in de toekomst interactie vloeiender, logischer en leuker kan maken.

Interface & Interactie (HTML, CSS & JavaScript)

De opdracht voor dit vak was het maken van een unieke Nintendo tijdlijn. Mijn concept ontstond doordat ik toevallig een oude la open had gemaakt en mijn oude controllers van verschillende PlayStation generaties zag liggen. De oudste controllers lagen achterin de la - de nieuwste (wel kapot) lagen voorin. Een unieke tijdlijn eigenlijk. Voor dit concept geldt hetzelfde als het concept van Ontwerpen 1: het bedenken van een concept hoeft niet altijd achter je bureau te gebeuren.

Ik vond het een leuk vak en ook een hele leuke opdracht. Ik had vorig jaar het vak Frontend voor Designers gevolgd, dus ik wist wel al wat van CSS af. Echter was het wel alweer bijna een jaar geleden, dus ik moest er wel weer even inkomen. Ik ben uiteindelijk blij met het visuele eindresultaat. Ik heb gekozen om alles te maken met CSS - het kostte me erg veel tijd, maar het geeft me uiteindelijk wel een veel beter gevoel. Illustraties maken in Illustrator voelde voor mij toch een beetje als 'valsspelen' bij dit vak. Doordat ik alles met CSS heb gemaakt heb ik ook weer meer geleerd over het maken van voorwerpen/vormen. Ook heb ik geleerd hoe ik geluid met HTMl & JavaScript kan schrijven. Dit wist ik tot heden niet, maar wilde ik wel altijd al graag leren. Ook is het gelukt om met JavaScript de buttons werkend te krijgen en heb ik geleerd dat ik met transform:translateY veel soepelere animaties kan maken.
